## 项目的功能点：

1、分页

2、数据校验（Jquery前端校验+JSR303后端校验）

3、ajax

4、Rest风格的URI：使用HTTP协议请求方式的动词，来表示对资源的操作（GET(查询)、POST(新赠)、PUT(修改)、DELETE(删除)）

## 技术点：

1、基础框架-ssm(SpringMVC+Spring+Mybatis)

2、数据库-MySQL

3、前端框架-bootstrap快速搭建简洁美观的界面

4、项目的依赖管理-Maven

5、分页-pagehelper

6、mybatis Generator-逆向工程
